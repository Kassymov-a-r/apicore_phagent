Инструкция

Render: 1. Deploy as Docker Web Service. 2. Add persistent disk mounted to /app/storage. 3. Open dashboard. 4. Add account. 5. In account card enter Instagram username/password and optional 2FA code. 6. Click "Войти и сохранить сессию". 7. Create automation rule. 8. Start agent. Пароли не сохраняются. Сохраняются только cookies/session в /app/storage.
OPENAI_API_KEY добавляется в .env, если хочешь AI-генерацию текстов.
