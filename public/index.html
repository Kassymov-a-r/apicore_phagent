Автоматизации
Сценарии ответов
Разделены две логики: комментарий → публичный ответ/Direct и входящее Direct-сообщение → автоответ.

        

      

      

        

          
Новый сценарий

          
Начни с одного короткого ключевого слова. Для Reels лучше явно писать в CTA: «Напишите АПИКОР в комментариях».

          
          
          
          
Каждое ключевое слово с новой строки. Проверка нечувствительна к регистру: «АПИКОР» и «апикор» совпадут.

          

            
            
            
          

          

            
Сценарий для комментариев. Пользователь пишет ключевое слово под постом/Reels. Бот может оставить публичный ответ и/или отправить первый приватный ответ в Direct, если Meta отдаёт webhook.

            
            
Добавь 2–4 варианта, чтобы ответы выглядели естественно. Не делай обещаний, которые бот не сможет выполнить.

            

          

          

            
Сценарий для Direct. Пользователь пишет ключевое слово в личные сообщения. Бот отвечает одним из вариантов ниже.

            
            
Лучше начинать с короткого человеческого сообщения. Ссылки добавляй аккуратно: Instagram может ограничивать доставку слишком рекламных сообщений.

            

          

          

            
            

            

          

          

          

        

        

          
Подсказки по сценариям

          

            
1
Чёткий CTA
В посте проси конкретное слово: «Напишите АПИКОР — отправлю ссылку».

            
2
Один смысл — один сценарий
Не смешивай «цена», «ссылка», «консультация» в одном длинном правиле.

            
3
Сначала тест
Перед реальным запуском проверь keyword matching через вкладку «Тест и предпросмотр».
