Аккаунты

Пароль не хранится. Вход выполняется через встроенный серверный браузер.

Remote Browser

Кликай по изображению как по браузеру. Для ввода текста нажми в поле на скрине, затем печатай ниже и отправляй в серверный браузер.

Как войти

  1. Нажми «Открыть Instagram».
  2. Кликни по полю логина на скрине.
  3. Введи логин через поле под скрином.
  4. Так же введи пароль и 2FA.
  5. После входа нажми «Сохранить сессию».
