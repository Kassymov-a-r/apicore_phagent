Setup: Instagram Integration

Что реально возможно

Официально есть два нормальных варианта: Instagram Login API для новых интеграций и Facebook Login fallback для старого Page-flow. Для публичных клиентов нужны Live Mode/App Review. Для своих тестовых аккаунтов можно работать в Development, если аккаунты добавлены как tester и permissions доступны.

Секреты / настройки

Здесь можно ввести все нужные данные: App ID, App Secret, Verify Token, OpenAI Key, APP_BASE_URL и другие параметры. Значения сохраняются в базе и сразу используются кодом.

Если поле показывает source = built-in, значит оно уже зашито в проект. Если хочешь заменить — просто введи новое значение и нажми Сохранить.
